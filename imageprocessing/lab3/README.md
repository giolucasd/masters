# Rodando o programa

## Linha de comando

Para rodar o programa, basta instalar as depedências:

```sh
pip install -r requirements.txt
```

e utilizar a linha de comando:

```sh
python alinhar.py inputs/sample1.png hough outputs/sample1.png
```
